The site is running on Flask using Jinja2 for templating. The timeline is fetched using requests and then stored in memcached using pylibmc. The frontend is based on Bootstrap from Twitter, inspired by the design of kristian.io. The app is proxied through nginx via gunicorn running in supervisor on a Ubuntu server on rackspace.
You can have a look at the source code of this site over at github.
